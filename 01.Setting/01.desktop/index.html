Dockerの構築

isaac SDKをダウンロードし、PCに展開します。

1
cd ~/isaac/sdk

1
../engine/engine/build/docker/install_docker.sh

1
../engine/engine/build/docker/create_image.sh

1
docker run --mount source=isaac-sdk-build-cache,target=/root -v `pwd`:/src/workspace -w /src/workspace --runtime=nvidia -it isaacbuild:latest /bin/bash

1
bazel build ...
